## Getting Started

To run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/RoselynNgito/green-reuse.git
   cd green-reuse


2. **Install dependencies for both frontend and backend**

   ```bash
   cd frontend
   npm install
   cd ../backend
   npm install
   ```

3. **Run the project**

   * Frontend

     ```bash
     cd frontend
     npm run dev
     ```
   * Backend

     ```bash
     cd backend
     npm start
     ```
